# T264: Research Tool Config Coverage & Document Capabilities

> **Priority**: P1 | **Wave**: 3 | **Track**: F (Generic Format Skills)
> **Depends on**: T231
> **Estimated complexity**: Low (research task)

## 1. Objective

Research each platform/tool config format (Cursor, Claude, Factory, MCP, ai-tool-sync) from official documentation. Produce per-tool analysis documents listing options, required fields, defaults, and identify gaps in current ai-tool-sync support.

## 2. Deliverables

### 2.1 Analysis Documents

Create `docs/analysis/` directory with:
- `cursor-config.md` - Cursor configuration analysis
- `claude-config.md` - Claude Code configuration analysis
- `factory-config.md` - Factory configuration analysis
- `mcp-config.md` - MCP specification analysis
- `capability-matrix.md` - Cross-platform comparison matrix

### 2.2 Document Template

    # [Platform] Configuration Analysis

    ## Official Sources
    - [Link to official docs]

    ## Supported Config Files
    | File | Format | Purpose |

    ## Configuration Options
    | Option | Type | Required | Default | Notes |

    ## ai-tool-sync Coverage
    | Feature | Supported | Notes |

    ## Gaps & Missing Features
    - [ ] Feature X - Priority: High/Medium/Low

### 2.3 Capability Matrix

    | Feature | Cursor | Claude | Factory | ai-tool-sync |
    |---------|--------|--------|---------|--------------|
    | Commands | Yes | Yes | Yes | Yes |
    | Hooks | Yes | Yes | Yes | Yes |
